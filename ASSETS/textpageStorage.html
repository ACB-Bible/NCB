New Christian Bible - About

The New Christian Bible is part of the Bible Code Project which is an ongoing Bible development project with multiple goals. Some of these goals include a Bible website for Bible study and Sunday services, and it can be used with many different devices. The Bible Code Project contains source code and data files that are in the public domain, and none of it is copyrighted. It's free to use for churches, organizations, and individuals in their services, Bible study groups, or even their own own Bible projects for free. God is not a business and the Bible is not a product, so all of this is free for you to use, and it's not for sale.



The Bible Code Project

Bible Code Project Documentation for Developers
If you'd like to try your hand at developing your own Bible website, the Bible Code Project is a collection of GitHub repositories containing source code and data that anyone can use to develop their own computerized Bible project. All of the data and source code for each of these projects is in the public domain and free of copyrights, therefore it's free for you to use. If you use or change any of the source code or data files in the Bible Code Project for your own ministry or Bible project, please change the name of your project, so people don't get confused.
